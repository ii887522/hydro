## Prerequisites
- [Docker Desktop](https://www.docker.com/products/docker-desktop) using Linux containers
- [Visual Studio Code](https://code.visualstudio.com/)
  - Docker
  - EditorConfig for VS Code
  - Markdown All in One
  - Remote - WSL
  - StandardJS - Javascript Standard Style
  - YAML
